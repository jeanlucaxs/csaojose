# Colégio São José - Site Institucional

Este é um site institucional para o Colégio São José, localizado em Prudentópolis - PR. O site foi desenvolvido como uma réplica do design e estrutura do site oficial [csjoficial.com.br](https://csjoficial.com.br), utilizando HTML, CSS (Tailwind CSS) e JavaScript básico.

## Funcionalidades

- **Header responsivo** com menu dropdown para Institucional e Ensino
- **Botão "Portal do Aluno"** no header, conforme solicitado
- **Seção Hero** com imagens animadas de estudantes
- **Seções informativas**:
  - Nossos Segmentos (Educação Infantil, Fundamental, Médio)
  - Nossos Diferenciais (Educação Socioemocional, Sistema COC, Gênio das Finanças, Maralto)
  - Notícias
  - Conquistas (galeria de imagens)
- **CTA para contato** com WhatsApp
- **Footer completo** com links, contato e redes sociais

## Assets

Os assets do site original foram baixados e estão organizados na pasta `original_assets/`:

- `logo.svg` - Logomarca do colégio
- `aluno-01.webp`, `aluna-02.webp`, `aluno-03.webp` - Imagens do hero section (desktop)
- `mobile_aluno-01.webp`, `mobile_aluna-02.webp`, `mobile_aluno-03.webp` - Imagens do hero section (mobile)
- `01.webp` a `12.webp` - Galeria de conquistas
- `Xadrez.webp`, `aluno-leitura.webp`, `feira-ciencias.webp` - Imagens de notícias
- `colegio.webp` - Imagem da fachada para CTA

## Como executar

1. Clone o repositório
2. Abra o `index.html` em um navegador web
3. Ou execute um servidor local:
   ```bash
   python3 -m http.server 8000
   ```
   Acesse em `http://localhost:8000`

## Tecnologias utilizadas

- HTML5
- CSS3 (Tailwind CSS via CDN)
- JavaScript (para animações básicas)
- SVG para ícones

## Responsividade

O site é totalmente responsivo, adaptando-se a dispositivos móveis e desktop.
